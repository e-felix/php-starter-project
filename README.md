# TINY ADSERVER

> A PHP 7.4 environment to start a project

## REQUIREMENTS

To install, you need the following programs:
 - docker
 - docker-compose

## BUILD DEV ENVIRONMENT

```sh
docker-compose up
```

## ATTACH TO SERVICE

```sh
docker-compose exec [service] bash
```

## SERVER ENDPOINT

```sh
http://locahost:8080
```
